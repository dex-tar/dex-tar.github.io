This script will check for:
1. Reboot required (based on the presence of /var/run/reboot-required file).
2. Disk usage for the root directory.
3. Disk usage for the root partition.
4. Network connectivity by attempting to ping Google.

Save this script with a .sh extension (e.g., health_check.sh) and make it executable
