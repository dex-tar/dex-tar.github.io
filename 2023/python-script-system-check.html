This Python script is designed for system administrators and users to perform health checks on their systems.

The script consists of functions to check various aspects of system health.

1. Checks if a reboot is pending.
2. Verifies if there is enough free disk space.
3. Specifically checks if the root partition is nearing full capacity.
4. Determines if there is an active network connection.
